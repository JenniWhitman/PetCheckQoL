# Pet Check QoL

A lightweight World of Warcraft addon that shows a big on-screen reminder when you’re on a pet-using class/spec and forgot to summon your pet.

Built for the post-WeakAura-chaos era, when forgetting your pet suddenly became a lifestyle.

## Features

- Big on-screen missing-pet warning text
- Moveable warning text with saved position
- Class/spec-aware tracking:
  - **Warlock** (all specs)
  - **Hunter** (BM/Survival by default, optional Marksmanship)
  - **Death Knight** (Unholy by default, optional Blood/Frost)
- In-game settings panel (`/petcheck options`)
- Editable warning text
- Font dropdown (built-in WoW fonts)
- Custom font support (add your own `.ttf` and register it)
- Font size controls
- Text color + shadow color controls
- Built-in color pickers
- Slash commands for testing, status, and config
